# Report template

A complete water-quality review report should include:

1. Review boundary and data provenance
2. Sensor coverage summary
3. Anomaly counts by station and parameter
4. Contamination pattern examples
5. Treatment-risk table
6. Alert-priority signal table
7. Regional and plant-level summary
8. Human review checklist
9. Limitations and next validation steps
10. Audit-log verification status
